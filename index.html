自動安全警示車

產品理念

當發生交通事故時，系統會自動啟動安全警示流程。裝置將自動往後移動至安全距離，升起警示牌，並將事故資料與即時影像傳送到後端監控系統。

透過 GPS 定位、視覺辨識與自動化控制，確保事故現場能夠快速建立安全警示，保護用路人安全。

系統功能

1. 自動定位

透過 GPS 模組自動取得事故位置，並計算移動距離。

2. 智能移動

根據道路速限自動判斷目標距離，並在移動過程中進行障礙物偵測與避障。

3. 視覺辨識

使用攝影機進行即時影像分析，偵測障礙物與人員，判斷是否有民眾受傷。

4. 警示系統

到達目標位置後自動升起警示牌，LED 燈條自動啟動，提供視覺警示。

5. 資料上報

自動將事故位置、時間、影像與傷亡資訊上報至後端監控系統。

6. 即時監控

提供即時影像串流，讓後端系統能夠遠端監控事故現場狀況。

操作說明

啟動系統

在樹莓派上執行主程式，系統將自動完成以下流程：

  1. 初始化 GPS 模組並取得事故位置
  2. 啟動視覺辨識模組
  3. 根據道路速限判斷目標距離
  4. 開始往後移動至安全距離
  5. 到達目標位置後升起警示牌
  6. LED 燈條自動啟動
  7. 上報事故資料至後端系統
  8. 開始提供即時影像串流

距離設定

系統根據道路速限自動判斷移動距離：

  • 高速公路（速限 ≥ 100 km/h）：100 公尺
  • 快速道路（速限 > 60 km/h）：80 公尺
  • 城市道路（速限 50-60 km/h）：50 公尺
  • 地方道路（速限 < 50 km/h）：30 公尺

監控系統

後端監控系統提供以下功能：

  • 地圖顯示事故位置
  • 即時影像串流
  • 事故列表管理
  • 管理員登入與資料管理

技術規格

控制平台

Raspberry Pi 4

定位系統

GPS NEO-M8 模組

視覺辨識

OpenCV + YOLO

馬達控制

L298N 驅動模組

警示系統

伺服馬達 + LED 燈條

通訊協定

HTTP REST API

注意事項

  • 使用前請確認所有硬體連接正確
  • 確保 GPS 模組能夠正常定位
  • 檢查攝影機連接與影像品質
  • 確認後端伺服器連線正常
  • 移動過程中請保持安全距離
  • 系統運行時請勿手動干擾
